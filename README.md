# Local Notebook LLM

A simple local application to interact with documents using a language model. Users can upload files and ask questions based on their content through a clean interface.

## Features

* Upload and query documents
* Context-based responses
* Local data processing (no external storage)
* Simple UI using Gradio

## Tech Stack

Python, Gradio, FAISS, Docker

## How to Run

```bash
git clone https://github.com/codecraft-777/local-notebooklm.git
cd local-notebooklm
pip install -r requirements.txt
python app.py
```
